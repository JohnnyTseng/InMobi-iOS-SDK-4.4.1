===================================
InMobi Monetization SDK 4.x for iOS
===================================

This is the InMobi Monetization SDK 4.x for iOS

Requirements:
- An InMobi app ID / site ID.
- Xcode 4.5 or later.
- Runtime of iOS 4.3 or later.

The latest documentation and code samples are available at:
http://www.inmobi.com/support/art/25447346/22404896/sdk-ios-integration-guide/
